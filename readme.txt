The YouTube Chromeless plugin allows one easily insert YouTube videos into WordPress posts and play it in customizing player.

== Description ==

The YouTube Chromeless plugin (developed by [CyberSEO.net](http://www.cyberseo.net/ "CyberSEO.net") and based on the [Custom YouTube Video Player](http://tutorialzine.com/2010/07/youtube-api-custom-player-jquery-css/ "Custom YouTube Video Player") tutorial by TutorialZine.com) allows one to play YouTube videos in the Chromless player. Use the following tag style to insert the YouTube videos into your posts:

        [youtube:url width]


where:

* url - URL of the YouTube video you want to embed;
* width - width of the chromeless player (height is being calculated automatically).

== Installation ==

1. Upload 'wp-flash.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
